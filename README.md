# Hotel Reservation System

## Description

The **Hotel Reservation System** is a Java-based console application developed using Object-Oriented Programming (OOP) concepts. This project allows users to search available hotel rooms, book rooms, cancel reservations, view booking details, and simulate payments. Booking information is stored permanently using Java File Handling, making the application simple, efficient, and user-friendly.

---

## Features

* View Available Rooms
* Search Rooms
* Book Hotel Rooms
* Cancel Reservations
* View Booking Details
* Payment Simulation
* File Handling using `bookings.txt`
* Room Categories (Standard, Deluxe, Suite)
* Menu-Driven Console Interface
* Object-Oriented Programming (OOP)

## How to Run

1. Clone the repository.

```bash
git clone https://github.com/sameerraj3479/HotelReservationSystem.git
```

2. Open the project in VS Code.

3. Run `Main.java`.

4. Use the menu to:

   * View Available Rooms
   * Book a Room
   * Cancel Booking
   * View Booking Details
   * Exit
